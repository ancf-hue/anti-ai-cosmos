> Офлайн-навигатор по звёздному небу для городов России

![Logo](logo.jpeg)

## ✨ Что умеет приложение

- 🏙️ **300+ городов России** — от Москвы до Петропавловска-Камчатского
- 🗣️ **5 языков** — русский, татарский, чеченский, башкирский, чувашский
- ✨ **20 созвездий** с анимацией и названиями звёзд
- 📷 **Фото и видео** неба с наложенными созвездиями
- 🔄 **Работает офлайн** — без интернета и без искусственного интеллекта

## 🚀 Как использовать

1. Откройте [сайт](https://ancf-hue.github.io/anti-ai-cosmos/)
2. Выберите свой город
3. Нажмите «🎬 Показать анимацию созвездия»
4. Смотрите, как звёзды соединяются в созвездия!

## 📱 Технологии

- HTML5 + CSS3 + JavaScript (Vanilla)
- Canvas API для анимации
- LocalStorage для сохранения настроек
- Адаптивный дизайн (мобильные устройства)
